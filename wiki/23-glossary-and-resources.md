## Glossary of key German terms

| Term | Meaning |
|------|---------|
| Anmeldung | Mandatory address registration at the Bürgeramt |
| Ummeldung / Abmeldung | Re-registration when you move / de-registration when you leave |
| Meldebescheinigung | Certificate of registration you receive after Anmeldung |
| Wohnungsgeberbestätigung | Landlord's confirmation that you live at the address |
| Bürgeramt / Einwohnermeldeamt | Citizens' office where you do Anmeldung |
| Ausländerbehörde | Foreigners'/immigration office (residence permits) |
| Aufenthaltstitel | Residence permit (umbrella term) |
| Aufenthaltserlaubnis | Temporary residence permit |
| Niederlassungserlaubnis | Permanent settlement permit |
| Fiktionsbescheinigung | Interim certificate bridging visa expiry until a permit is issued |
| eAT | Electronic residence-permit card |
| Untätigkeitsklage | Lawsuit for an authority's failure to act (§75 VwGO) |
| EU Blue Card / Blaue Karte EU | Skilled-worker permit for degree-holders |
| Chancenkarte | Opportunity Card — points-based job-seeker route |
| Fachkräfteeinwanderungsgesetz | Skilled Immigration Act |
| Steuer-ID / Steueridentifikationsnummer | Permanent 11-digit tax ID |
| Steuernummer | Tax number (freelancers/business) |
| Steuerklasse | Tax class (I–VI) |
| Steuererklärung | Annual tax return |
| Sozialversicherungsnummer | Social-security/pension number |
| GKV / PKV | Public / private health insurance |
| Gesundheitskarte | Health insurance card |
| Hausarzt | GP / family doctor |
| Krankenkasse | Health insurance fund (TK, AOK, Barmer…) |
| Schufa | Main credit-scoring agency |
| Girokonto | Current/checking bank account |
| Lastschrift / Dauerauftrag | Direct debit / standing order |
| Kaltmiete / Warmmiete | Cold rent (base) / warm rent (with utilities) |
| Nebenkosten | Utility/service charges |
| Kaution | Rental deposit (max 3 months' cold rent) |
| Selbstauskunft | Tenant self-disclosure form |
| WG / Wohngemeinschaft | Shared flat |
| Anerkennung | Recognition of foreign qualifications |
| anabin | Official database of foreign degrees/institutions |
| ZAB / Zeugnisbewertung | Central office / Statement of Comparability for degrees |
| Approbation | Licence to practise (doctors, pharmacists…) |
| Kenntnisprüfung / Fachsprachprüfung | Knowledge exam / professional-language exam |
| Anerkennungspartnerschaft | Recognition Partnership (work while recognition pending) |
| Integrationskurs | State-funded integration (language + orientation) course |
| Berechtigungsschein | Entitlement voucher for the integration course |
| DTZ / "Leben in Deutschland" | Language test / life-in-Germany test |
| Bewerbung | Job application (whole package) |
| Lebenslauf | CV / résumé |
| Anschreiben | Cover letter |
| Arbeitszeugnis | Coded employer reference letter |
| Arbeitsvertrag | Employment contract |
| Rundfunkbeitrag | Mandatory public-broadcasting fee (per household) |
| Pfand | Bottle/can deposit |
| Ruhezeit / Ruhetag | Legal quiet hours / rest day (Sundays) |
| Haftpflichtversicherung | Personal liability insurance |
| Bürgergeld | Basic state welfare benefit |
| Mutterschutz | Statutory maternity protection period |
| Elterngeld | Parental allowance (income replacement) |
| Elternzeit | Parental leave (unpaid job-protected leave) |
| Kindergeld | Child benefit |
| Geburtsurkunde | Birth certificate |
| Standesamt | Civil registry office (births, marriages) |
| eVB-Nummer | Electronic insurance confirmation number, needed to register a car |
| Kfz-Zulassungsstelle / Zulassungsbehörde | Vehicle registration office |
| TÜV / Hauptuntersuchung (HU) | Mandatory periodic vehicle inspection |
| Umweltzone / Umweltplakette | Low-emission zone / emissions sticker |
| Termin | Appointment |
| Bundesland | Federal state (rules vary by state) |

## Official & trusted resources

**Government / official**
- [Make it in Germany](https://www.make-it-in-germany.com/en/) — the official portal for skilled workers
- [BAMF](https://www.bamf.de/EN/) (Federal Office for Migration and Refugees)
- [Recognition in Germany](https://www.anerkennung-in-deutschland.de/html/en/) (Anerkennung portal)
- [anabin](https://anabin.kmk.org/) (degree database)
- [Federal Foreign Office](https://www.auswaertiges-amt.de/en) (visas)
- [Bundesagentur für Arbeit](https://www.arbeitsagentur.de/en) (Federal Employment Agency / job board)
- [Stadt Nürnberg](https://www.nuernberg.de/) — official city portal, gateway to Bürgeramt, Ausländerbehörde, Führerscheinstelle, Standesamt, and Zulassungsbehörde services referenced throughout this wiki

**Practical guides**
- [Handbook Germany](https://handbookgermany.de/en) — multilingual newcomer guide
- [liveingermany.de](https://liveingermany.de) — the primary source used throughout this wiki
- IamExpat, The Local Germany — English-language news & how-tos
- Doctolib / Jameda — book doctors (filter by language & insurance)
- ImmoScout24 / WG-Gesucht / Immowelt — housing
- [Antidiskriminierungsstelle des Bundes](https://www.antidiskriminierungsstelle.de/EN) — federal anti-discrimination agency

> ⚠️ Rules vary by Bundesland and city, and changed repeatedly in 2023–2026. Always confirm with your local Ausländerbehörde and the official portals above before acting.

## Quick reference: what unlocks what

```
Anmeldung ──► Meldebescheinigung ──┬──► Tax ID ──► correct payroll
                                   ├──► Residence permit
                                   ├──► Full bank account
                                   └──► Health insurance, contracts
Recognised qualification ──► regulated jobs + less deskilling
B1 German ──► faster permanent residence + citizenship
```
