# Craftpix asset attribution

`house.png` is a cropped, nearest-neighbor 2:1 version of the ready-made house at the top-left of `PNG/exterior.png` from the Craftpix "Main Characters Home" free top-down pixel-art asset package.

Source: https://craftpix.net/freebies/

The curated Craftpix tree pack used by the world is documented separately in [`trees/README.md`](trees/README.md), alongside its exact supplied license file.
